# AlexARICHITECHTURE

AlexNet is a popular convolutional neural network architecture that won the ImageNet 2012 challenge by a large margin. 
It was developed by Alex Krizhevsky, Ilya Sutskever and Geoffery Hinton. It is similar to the LeNet-5 architecture but larger and deeper.
